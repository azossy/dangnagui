1. 프로그램 소개
────────────────────────────────────────────────────────────

  게시판 검색기는 사용자가 관심 있는 토픽(주제)에 대해
  DuckDuckGo 실시간 웹 검색을 통해 핫키워드와 네티즌 의견을
  수집하여, 메신저 스타일 리포트를 생성하는 프로그램입니다.

  주요 기능:
    • 9개 기본 토픽 + 사용자 정의 토픽 무제한 추가
    • DuckDuckGo text + news 실시간 검색
    • 토픽별 핫키워드 중요도 순 정렬
    • 네티즌 의견 요약 제공
    • 메신저 스타일 리포트 자동 생성
    • 클립보드 복사, 파일 저장, 소셜 미디어 공유


2. 시스템 요구사항
────────────────────────────────────────────────────────────

    • Windows 10 이상 (64비트)
    • 화면 해상도 1280x720 이상 권장
    • 인터넷 연결 필수 (실시간 웹 검색)


3. 설치 방법
────────────────────────────────────────────────────────────

  ■ 인스톨러로 설치 (권장)

    1. dangnagui-setup-v1.2.3.exe를 다운로드합니다.
       → https://github.com/azossy/dangnagui/releases/latest

    2. 다운로드한 파일을 더블클릭하여 실행합니다.
       ※ Windows SmartScreen 경고가 뜨는 경우:
         "추가 정보" → "실행" 을 클릭하세요.

    3. 설치 경로를 선택합니다.
       (기본: C:\Program Files\dangnagui)

    4. "설치" 버튼을 클릭합니다.

    5. 설치 완료 후:
       - 바탕화면의 "당나귀 게시판검색기" 아이콘 더블클릭
       - 또는 시작 메뉴 → "당나귀 게시판검색기" 클릭

    ※ 삭제 방법:
       설정 → 앱 → "당나귀 게시판검색기" → 제거
       또는 시작 메뉴 → "당나귀 게시판검색기" 폴더 → 제거


  ■ 포터블(USB) 설치 (설치 없이 사용)

    1. dangnagui-v1.2.3-portable.zip을 다운로드합니다.
       → https://github.com/azossy/dangnagui/releases/latest

    2. 원하는 위치에 압축을 해제합니다.
       - USB 드라이브, 바탕화면, 외장하드 등 어디든 가능

    3. 압축 해제된 폴더 안의 dangnagui.exe를 실행합니다.

    ※ 설정/로그 파일이 exe와 같은 폴더에 저장됩니다.
    ※ 폴더째 복사하면 설정이 유지됩니다.
    ※ 삭제 시 폴더만 지우면 됩니다 (레지스트리 사용 안 함).


4. 사용 방법
────────────────────────────────────────────────────────────

  [기본 사용]

    1. 프로그램을 실행합니다.
    2. "▶ 스타트" 버튼을 클릭합니다.
    3. 토픽별 DuckDuckGo 실시간 검색이 진행됩니다.
       (프로그레스 바로 진행률 표시)
    4. 완료되면 토픽별 핫키워드 리포트가 생성됩니다.
    5. "📋 복사" 버튼으로 클립보드에 복사하거나
       "💾 저장" 버튼으로 파일로 저장할 수 있습니다.

  [설정 변경 — ⚙ 아이콘 클릭]

    • 검색 기준 시간: 현재 시간 기준 몇 시간 이전까지의
      게시판 글을 검색 대상으로 할지 설정 (30~100시간)

    • 토픽 관리:
      - 클릭: 토픽 활성/비활성 토글 (파란색 = 활성)
      - 드래그: 토픽 순서 변경
      - ✕ 버튼: 토픽 삭제
      - "핫키워드 N개": 해당 토픽에서 추출할 핫키워드 수 (1~10)

    • 새 토픽 추가: 토픽명을 입력하고 "추가" 클릭
      → 웹검색으로 관련 사이트/게시판을 자동 탐색 후 등록

    • 사이트 갱신: 모든 토픽의 관련 사이트를 최신화
      → "전체 초기화 후 새로 구축"
      → "기존 유지 + 새 사이트만 추가"

    • 💾 설정 저장 (Ctrl+S): 변경사항 저장
    • 닫기 (Esc): 설정 창 닫기

  [리포트 공유]

    • 소셜 아이콘 (Facebook, X, 카카오톡 등):
      리포트를 클립보드에 복사한 후 해당 앱을 실행합니다.
      앱에서 붙여넣기(Ctrl+V)로 리포트를 공유하세요.


5. 파일 구성
────────────────────────────────────────────────────────────

    dangnagui.exe           프로그램 실행 파일
    dangnagui.ico           프로그램 아이콘
    app_settings.json       사용자 설정 (자동 생성)
    readme.txt              이 파일
    IMoutput\               리포트 저장 기본 폴더
    logs\                   로그 파일 (문제 발생 시 참고)


6. 문제 해결
────────────────────────────────────────────────────────────

    Q: 검색 결과가 없다고 나와요.
    A: 인터넷 연결을 확인하세요. DuckDuckGo 실시간 검색에
       인터넷이 필수입니다.

    Q: SmartScreen 경고가 떠요.
    A: "추가 정보" → "실행"을 클릭하세요.
       오픈소스 프로그램이라 코드 서명이 없어 나타나는 정상 경고입니다.

    Q: 설정이 이상해졌어요.
    A: app_settings.json 파일을 삭제하면 기본값으로 초기화됩니다.
       (.bak 백업 파일로 복원도 가능)

    Q: 프로그램이 두 번 실행됐다는 경고가 나와요.
    A: 이미 실행 중인 프로그램 창을 확인하세요.
       동시 실행 시 설정 파일이 손상될 수 있습니다.
