1. 프로그램 소개
────────────────────────────────────────────────────────────

  게시판 검색기는 사용자가 관심 있는 토픽(주제)에 대해
  국내 주요 사이트와 게시판을 실시간으로 검색하여,
  핫키워드와 네티즌 의견을 한눈에 파악할 수 있는 프로그램입니다.

  주요 기능:
    • 9개 기본 토픽 + 사용자 정의 토픽 무제한 추가
    • 3,000개 이상의 사이트, 33,000개 이상의 게시판 검색
    • 토픽별 핫키워드 중요도 순 정렬
    • 네티즌 의견 요약 제공
    • 메신저 스타일 리포트 자동 생성
    • 클립보드 복사, 파일 저장, 소셜 미디어 공유


2. 시스템 요구사항
────────────────────────────────────────────────────────────

    • Windows 10 이상 (64비트)
    • 화면 해상도 1280x720 이상 권장
    • 인터넷 연결 (사이트 갱신 시 필요)


3. 설치 및 실행
────────────────────────────────────────────────────────────

  [인스톨러 사용 시]
    1. dangnagui_setup.exe 실행
    2. 설치 경로 선택 (기본: C:\Program Files\dangnagui)
    3. 설치 완료 후 바탕화면 또는 시작 메뉴에서 실행

  [포터블 사용 시]
    1. dangnagui 폴더를 원하는 위치(USB 등)에 복사
    2. 폴더 내 dangnagui.exe 실행
    ※ 설정 파일이 실행 파일과 같은 폴더에 저장됩니다.


4. 사용 방법
────────────────────────────────────────────────────────────

  [기본 사용]

    1. 프로그램을 실행합니다.
    2. "▶ 스타트" 버튼을 클릭합니다.
    3. 사이트 검색 진행률이 프로그레스 바로 표시됩니다.
    4. 완료되면 토픽별 핫키워드 리포트가 생성됩니다.
    5. "📋 복사" 버튼으로 클립보드에 복사하거나
       "💾 저장" 버튼으로 파일로 저장할 수 있습니다.

  [설정 변경 — ⚙ 아이콘 클릭]

    • 검색 기준 시간: 현재 시간 기준 몇 시간 이전까지의
      게시판 글을 검색 대상으로 할지 설정 (30~100시간)

    • 토픽 관리:
      - 클릭: 토픽 활성/비활성 토글 (파란색 = 활성)
      - 드래그: 토픽 순서 변경
      - ✕ 버튼: 토픽 삭제
      - "핫키워드 N개": 해당 토픽에서 추출할 핫키워드 수 (1~10)

    • 새 토픽 추가: 토픽명을 입력하고 "추가" 클릭
      → 웹검색으로 관련 사이트/게시판을 자동 탐색 후 등록

    • 사이트 갱신: 모든 토픽의 관련 사이트를 최신 상태로 업데이트
      → "전체 초기화 후 새로 구축" 또는 "기존 유지 + 새 사이트만 추가" 선택

    • 💾 설정 저장 (Ctrl+S): 변경사항 저장
    • 닫기 (Esc): 설정 창 닫기 (미저장 시 확인 창 표시)

  [리포트 공유]

    • 소셜 아이콘 (Facebook, X, 카카오톡 등):
      리포트를 클립보드에 복사한 후 해당 앱을 실행합니다.
      앱에서 붙여넣기(Ctrl+V)로 리포트를 공유하세요.

    • ✉ 이메일: 개발자에게 문의 메일을 보낼 수 있습니다.


5. 파일 구성
────────────────────────────────────────────────────────────

    dangnagui.exe           프로그램 실행 파일
    sites_config.json       검색 대상 사이트/게시판 데이터
    커뮤니티_핫주제_실시간.md   리포트 원본 데이터
    app_settings.json       사용자 설정 (자동 생성)
    readme.txt              이 파일
    IMoutput\               리포트 저장 기본 폴더
    logs\                   로그 파일 (문제 발생 시 참고)


6. 주의사항
────────────────────────────────────────────────────────────

    • 30일마다 "사이트 갱신"을 권장합니다. (프로그램 시작 시 알림)
    • 사이트 갱신에는 인터넷 연결이 필요합니다.
    • 설정 파일(app_settings.json)을 직접 편집하지 마세요.
    • 문제가 발생하면 logs 폴더의 로그 파일을 참고하세요.
    • 설정이 손상된 경우 app_settings.json 파일을 삭제하면
      기본값으로 초기화됩니다. (.bak 백업 파일로 복원도 가능)
